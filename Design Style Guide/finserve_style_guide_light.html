Do / don't
Do
Use red only for primary CTAs, urgency signals, and brand moments. One dominant red action per view.
Don't
Use red decoratively or for info states. Never place two primary red buttons side by side.
Do
Keep badge color-status pairings consistent. Same status = same color across every screen.
Don't
Use different labels or colors for the same state (e.g. "Active" vs "In progress" on different screens).
Do
Format all amounts with ₹ prefix and Indian notation: ₹62,00,000. Use one shared formatINR() util.
Don't
Mix formats: ₹62L vs ₹6200000 vs ₹62,00,000 across screens. Agree once, enforce everywhere.
